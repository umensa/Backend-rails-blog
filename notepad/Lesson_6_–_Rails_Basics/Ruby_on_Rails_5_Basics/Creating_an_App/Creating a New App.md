#Creating a New App

**Challenge Task 1 of 3**
Use this command line to create a new Rails app named vet.

	rails new vet

**Challenge Task 2 of 3**

Now change into the directory for your new vet app.

	cd vet

**Challenge Task 3 of 3**

Now that you're in the app directory, try running a web server.

	bin/rails server
